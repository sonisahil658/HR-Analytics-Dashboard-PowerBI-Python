# HR-Analytics-Dashboard-PowerBI-Python
Complete end-to-end Data Analytics project analyzing HR workforce data using Python and Power BI

# Project Overview

This project analyzes HR data of 22,213 employees across 13 departments and 7 US states to understand workforce composition, employee attrition patterns, gender and race diversity, and department-level performance. The analysis was performed using Python (Pandas) for data cleaning and Power BI for interactive dashboard development to generate actionable HR insights and support data-driven retention decisions.


# Business Problem Statement

Organizations face increasing challenges in managing employee retention, identifying high-risk departments, and understanding workforce diversity. A high attrition rate directly impacts productivity, recruitment costs, and team morale.

This project aims to analyze HR workforce data and identify patterns in employee termination, department-level attrition risk, gender and race distribution, location-based workforce trends, and job title turnover rates — helping HR leadership make informed decisions to improve employee retention and engagement.


# Project Deliverables

1. Data Preparation (Python)

Clean and transform raw HR data, handle missing values, derive the employee status (Active vs Terminated) from termination dates, calculate attrition rates, and prepare the dataset for analysis.

2. Visualization & Insights (Power BI)

Build an interactive dashboard to visualize workforce distribution, attrition by department, gender and race diversity, location-based employee counts, and active vs terminated comparison by department.

3. Insights Report

Document key findings, attrition risk analysis, and recommendations derived from the data.

4. GitHub Repository

Maintain all project files including Python scripts, Power BI dashboard, dataset, insights report, and project documentation.

# Key Insights


The overall attrition rate is 17.69% — above the industry benchmark of 15%.
Auditing has the highest attrition rate at 23.1% despite being the smallest department (52 employees).
Remote employees show lower attrition (16.7%) compared to Headquarters employees (18.0%).
Female employees have a slightly higher attrition rate (18.1%) vs Male (17.4%).
Financial Analysts and Systems Administrators face the highest attrition at 25% among all job titles.
Native Hawaiian / Pacific Islander employees have the highest attrition rate (19.4%) among all racial groups.
Marketing is the healthiest department with the lowest attrition at just 14.6%.



# Business Recommendations


Conduct immediate retention interviews and compensation benchmarking for the Auditing and Legal departments.
Expand remote and hybrid work options — data shows remote workers stay longer.
Perform a pay equity audit to investigate the female attrition gap.
Benchmark Financial Analyst and Systems Administrator salaries against industry standards.
Launch a succession planning program — 17.3% of the workforce is aged 51–60.
Conduct focused engagement surveys for the Native Hawaiian / Pacific Islander employee group.



Conclusion

This project provides a comprehensive analysis of a company's HR workforce using Python-based data preparation and Power BI visualization. The findings reveal critical attrition risks in specific departments and job roles, highlight diversity patterns across gender and race, and demonstrate the retention advantage of remote work — helping HR leadership prioritize resources and build a stronger, more engaged workforce.
